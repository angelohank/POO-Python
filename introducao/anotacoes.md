# Introdução

### Paradigmas de programação

- Procedural
    - Funções
    - Estruturas lineares de código

- Programação Orientada a Objetos
    - classes e objetos
    - estruturas paralelas


### Por que utlizar as classes?

- facilitam:
    - programação de sistemas complexos
        - maiores e mais amplos
    - manutenção
    - trabalho em equipe

- Comportamentos importantes:
    - instâncias múltiplas
    - Personalização por meio de herança
    - sobrecarga de operador (polimorfismo)
