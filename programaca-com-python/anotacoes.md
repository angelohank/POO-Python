# Programação com Python

### Tudo em python são objetos
- mais simples
  - tipagem dinâmica (diferente de C++, Java e C#, por exemplo)

em resumo: <br>
    `objeto.atributo` -> propriedade de um objeto <br>
    `objeto.metodo` -> ação que o objeto pode executar

### Sobre classes e objetos

**Classes**
- servem como fábrica de instâncias de objetos;
  - são projetos, como a planta de uma casa.
  - um mesmo projeto pode ser usado para construir várias casas iguais
    - a casa pronta, é a instância
  
- atributos fornecem caracteristicas;
- métodos fornecem comportamentos;

**Instâncias**
- representa os itens concretos no domínio do sistema
- depende do tipo de objeto para suas atribuições
